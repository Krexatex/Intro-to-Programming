HTML BASICS

Tags

Tags are the most basic component of HTML. There are opening and closing tags.
Examples:
the b tag "<b> content </b>". used to make the content bold
the em tag " <em> content </em> " to italicize a content.

Elements
Elements = opening tag + content + closing tag

HTML attributes
Some tags take additional information in the form of attributes that have values.
Examples:
The anchor tag "<a> content </a>" used to add links takes the attribe "href" with value the web adress of a site.
"<a href="udacity.com"> This is the syntax of a link to Udacity.com </a>"
It should be displayed this way: This is the syntax of a link to Udacity.com

The image "img" tag also takes attributes such as "src" which is the location of the image and "alt" for if the link is broken. It is required
"<img src="https://s3-us-west-1.amazonaws.com/udacity-content/images/icon-error.svg" alt="Broken Image?"> </a>"
